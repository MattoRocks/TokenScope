About TokenScope

This tool is free for the crypto-community to use, so don't abuse it or it'll go away! Consider it a free service for personal use, with no gaurantees or warantees.

I built this to create a better reading experience of token data, especially of a token's description content. Besides presenting description information in an easy-to-read space, TokenScope also renders any Markdown formatting that may be present (scope this link to see Markdown in action). Currently, this site uses OpenSea.io's API. As of version 0.8, this entire site is open source.

I'm Matto#7777 on Discord, AKA the artist Azurine (Website / Rarible), and the author M. Lorrox (Website / Rarible). I intend to make this site fully open-source in future updates, but currently, the repo linked below only contains the custom javascript I wrote (I'm new to this!).
