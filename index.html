About TokenScope

This tool is free for the crypto-community to use, so don't abuse it or it'll go away! Consider it a free service for personal use, with no gaurantees or warantees.

I built this to create a better reading experience of token data, especially of a token's description content. Besides presenting description information in an easy-to-read space, TokenScope also renders any Markdown formatting that may be present (scope this link to see Markdown in action).

This site uses OpenSea.io's API. As of version 1.0, this entire site is open source.
